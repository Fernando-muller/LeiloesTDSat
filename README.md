# 📌 Sistema de Leilões (LeiloesTDSat)

Sistema desktop desenvolvido em Java (Swing) com integração ao MySQL, criado como atividade avaliativa do curso do Senac.

# 🚀 Funcionalidades

Cadastro de produtos para leilão

Inserção de nome e valor do produto

Listagem de produtos cadastrados

Interface gráfica simples e funcional

# 🖥️ Tecnologias Utilizadas

Java (Swing)

MySQL

JDBC

NetBeans IDE
